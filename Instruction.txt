   1. На локальном репозитории сделать ветки:
- Student
- Mentor

Выполнение:
$ git branch Student
$ git branch Mentor


   2. Запушить все ветки на внешний репозиторий.
$ git push -u origin Student Mentor

   3. В ветке Student сделать текстовый документ c описанием выполненной задачи.
$ git checkout Student
$ cat > Simplified_testing_classification.txt
Здесь ввести текст.
Ctrl + D
Ctrl + C

   4. Запушить выполненную задачу на внешний репозиторий.
$ git add Simplified_testing_classification.txt
$ git commit -m "Add new file"
$ git push

   5. Вмержить ветку Student в Mentor.
$ git checkout Mentor
$ git merge Student

   6. Запушить Mentor на внешний репозиторий.
$ git push -u origin Mentor

   7. В ветке Student создать текстовый файл как моя работа.
$ git checkout Student
$ cat > how_my_homework.txt

   8. Запушить текстовый файл как моя работа на внешний репозиторий.
$ git add how_my_homework.txt
$ git commit -m "add new file"
$ git push


  9. На внешнем репозитории сделать Pull Request ветки Student в Mentor.

Открываем страницу репозитория.
Заходим во вкладку Pull requests.
Жмем Compare & pull request (выбираем строку с веткой Student).

base: выбираем Mentor
compare: выбираем Student

Смотрим на ветки, какой куда можно мержить на данный момент (Able to merge. These branches can be automatically merged.).
Оставляем комментарий.
Жмем Create pull request
Жмем Merge pull request
Жмем Confirm merge

  10. Синхронизировать Внешнюю и Локальную ветки Mentor
$ git checkout Mentor
$ git pull
